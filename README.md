CVSKit is a collection of Swift extensions and utility for better programming.

## Requirements
- **iOS** 10.0+
- Swift 4.2+


## Installation

CVSKit is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CVSKit'
```
